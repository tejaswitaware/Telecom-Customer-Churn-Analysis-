# Telecom Customer Churn Analysis 

## Overview

This project is an Excel-based dashboard that analyzes telecom customer data to identify churn patterns and customer behavior. The dashboard provides insights into churn rate, customer demographics, contract types, and internet services.

## Key Metrics

* Total Customers
* Total Churn
* Churn Rate
* New Joiners

## Features

* Churn analysis by category
* Churn by internet service type
* Contract type churn rate
* Customer churn by age group
* Tenure-based churn analysis
* Top cities with highest churn

## Tools Used

* Microsoft Excel
* Pivot Tables
* Data Visualization (Charts & Graphs)
